 # SETUP
 This project is a template DBT project, which includes macros and utilities to be used in generating models following the Slowly Changing Dimension Type2 RuleSet. 
 
 **This project is a template, do not write to any of the placeholders**

 To learn on how to use the template, visit this link: [Setup SCD DBT Template](https://dev.azure.com/BAE-UDO/Framework%20-%20Data%20Tools/_wiki/wikis/Framework---Data-Tools.wiki/1462/X.1-Project-Setup)
